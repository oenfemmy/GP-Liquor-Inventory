GitHub Sync Settings

Data disimpan sebagai inventory_data.json di GitHub repo kamu. Setiap kali kamu add/edit/delete item, file JSON di repo otomatis ke-update via GitHub API.

Setup:
1. Buat Personal Access Token di GitHub → Settings → Developer settings → Fine-grained tokens
2. Permission: Contents → Read and write
3. Scope ke repo yang kamu pakai untuk GitHub Pages
